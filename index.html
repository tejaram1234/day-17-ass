Location Object Model :

The Location object specifies the URL currently being displayed, and allows JavaScript code to load new URLs

Different Usages Of Location Object Models:

location.assign();//loads new documents

location.relode();//reloades the current document

location.replace();/replaces the current document with new one.


History Objects Model:

The History object contains information about the URLs that have been previously displayed in the window.

Different Usages Of History Object Models:

history.back();//for previous page

history.forward();//for next page

history.go(2);//for next 2nd page

history.go(-2);//for previous 2nd page

Navigator Object Model:

The navigator object is a part of the browser object model (BOM) that provides information about the browser environment and capabilities to the web page or script. It represents the browser window and can be used to retrieve information about the browser, such as the browser name, version, and operating system.

Example:

Here's an example of how to use the navigator object in JavaScript to detect the user's browser and operating system:

// Check if the user is using Internet Explorer

if (navigator.userAgent.indexOf('MSIE') !== -1 || navigator.appVersion.indexOf('Trident/') > 0) {

alert('You are using Internet Explorer.');

}

// Check if the user is using Mac OS X

if (navigator.platform.indexOf('Mac') !== -1) {

alert('You are using Mac OS X.');

}
